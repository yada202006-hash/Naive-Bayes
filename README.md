# Naive Bayes Classifier

## 📌 Overview
This project implements the **Naive Bayes algorithm** for classification tasks.  
Naive Bayes is a probabilistic classifier based on Bayes’ Theorem with the assumption of independence between features.  
It is simple, fast, and effective for text classification, spam filtering, and sentiment analysis.

---

## ⚙️ Algorithm Variants
- **Gaussian Naive Bayes** → for continuous/numeric features (assumes normal distribution).
- **Multinomial Naive Bayes** → for discrete counts (word frequencies, text classification).
- **Bernoulli Naive Bayes** → for binary features (presence/absence of words).

---

## 📂 Dataset
- Input: Features (numeric or categorical depending on variant).
- Output: Target labels (classes).
- Example: Weather prediction dataset with features like temperature, humidity, and date.
